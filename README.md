# cola-blog 内容仓库

这是 cola-blog 的内容仓库，包含所有的博客文章、数据文件和图片资源。

本博客的完整代码仓库请访问：[https://github.com/Xxxhite/cola-blog](https://github.com/Xxxhite/cola-blog)

## 目录结构 (Directory Structure)

- `posts/` - 博客文章
- `spec/` - 特殊页面 (关于、友链等)
- `data/` - 数据文件 (番剧、项目、技能、时间线)
- `images/` - 图片资源

## 使用说明 (Usage Instructions)

本项目参考并使用了 **Mizuki** 博客模板构建。关于本内容仓库如何与代码库关联，以及更详细的配置与使用方法，请直接查阅 Mizuki 原项目的官方使用说明：

[https://github.com/matsuzaka-yuki/Mizuki](https://github.com/matsuzaka-yuki/Mizuki)
